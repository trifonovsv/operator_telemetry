collect_hardware_statistic.py-логирование телеметрии локальных ресурсов ПК. Работает на windows и Linux(Ubuntu)
        * Нагрузка ЦП
	* %Сводобной памяти
	* Сетевая нагрузка Upload/Download
Требования: Python>=3.9
            Pandas
            Psutil

Пример запуска в cmd:
	py collect_hardware_statistic.py  <имя папки> <имя сетевого интерфейса>    
	,где <имя сетевого интерфейса> - имя сетевого интерфейса нагрузку которого необходимо отслеживать
        <имя папки> имя папки где будут храниться данные. Предпочтительно указывать как "localhost" или IP адрес ПК для удобаства
        Данные сохраняются в директории где был запущен скрипт .\telemetry\<имя папки>\<дата> в отдельной папке за текущие сутки


collect_channel_record_statistic.py-Скрипт сбора информации о длительности ошибки записи по каждому каналу и количестве каналов со сбоями.
Требования: Python>=3.9
            Pandas
            requests 
Работает на windows и Linux(Ubuntu)
    Пример запуска: "py <имя файла>.py <IP адрес>
    <IP адрес> - адрес наблюдаемого сервера, он же имя папки в которой сохраняется БД (По умолчанию ".\telemetry\<имя папки>")
    Данные сохраняются в директории где был запущен скрипт .\telemetry<имя папки>\<дата> в отдельной папке за текущие сутки

get_hardware_statistic.py - Скрипт для построения графиков статистики потребления ресурсов 
Требования: Python>=3.9
            Pandas
            Matplotlib
    Пример запуска на Linux: "python3 <имя файла>.py <имя папки> <дата>"
    Пример запуска на Windows: "py <имя файла>.py <имя папки> <дата>"
    Где <имя папки> - имя папки где хранится БД с данными
    Где <дата> - дата (она же имя вложенной папки) за которую необходимо построить статистику

get_general_channel_record_statistic.py - Скрипт построения графика статиcтики записи каналов за сутки
